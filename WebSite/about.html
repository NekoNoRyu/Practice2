О проекте "ProgrevTech"

Концепция

"ProgrevTech" — это инновационный проект, реализованный в виде Telegram-бота @GoeGrevBot. Наша миссия — сделать финансовую грамотность доступной, понятной и увлекательной для широкой аудитории. Мы используем элементы игры, чтобы вовлечь пользователя в процесс обучения и самоанализа. Основной инструмент — интерактивный тест, который определяет психотип пользователя в мире финансов и предлагает индивидуальный путь развития.

Цели и задачи

Ключевые цели проекта:

  • Классификация: Разделить пользователей на четыре категории («Серая масса», «Гой», «Средний класс», «Будущий миллионер») на основе их ответов, чтобы предоставить релевантный контент.
  • Образование: Предложить каждой категории пользователей уникальные образовательные материалы: от списков литературы и статей до специализированных тестов и интерактивных новелл.
  • Мотивация: Стимулировать пользователей к развитию, предлагая им перейти на следующий уровень финансовой осознанности и открывая новые возможности внутри бота.
  • Партнёрство: Интегрировать реферальные ссылки партнёров (банков и брокеров) как логичный следующий шаг для финансово созревших пользователей.

Основные задачи:

  • Разработка Telegram-бота на Python с использованием фреймворка Aiogram.
  • Проектирование и создание базы данных на PostgreSQL для хранения информации о пользователях, вопросах и результатах.
  • Создание уникального контента: текстов, вопросов для тестов, сценария для интерактивной новеллы.
  • Реализация машины состояний (FSM) для управления диалогами и прохождением тестов.
  • Внедрение системы защиты от спама и фоновых задач для отправки напоминаний.

Ожидаемые результаты

По итогам реализации проекта мы ожидаем достичь следующих результатов:

  • Повышение уровня финансовой грамотности и вовлечённости у целевой аудитории.
  • Создание активного сообщества пользователей, стремящихся к финансовому росту.
  • Успешная модель монетизации через партнёрские программы, которая поддерживает дальнейшее развитие проекта.
  • Формирование уникального образовательного инструмента, который можно масштабировать и адаптировать под новые задачи.
